> 카운터 기능 커스텀

------------------------------------------------------------------------------------------

** 구글 머터리얼 디자인 사용법 **

Static icon font 의 링크 복사

SCSS에 @import url("https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@24,400,0,0");

html에 Inserting the icon 내용 복사 후 붙여넣기.
<span class="material-symbols-outlined">
remove
</span>

버튼 추가로 넣을때도 Inserting the icon 내용만 다시 넣는다.
<span class="material-symbols-outlined">
add
</span>

!! 주의점 span 안의 텍스트와 class를 바꾸면 안된다.


------------------------------------------------------------------------------------------

html에서 데이터 값을 지정하기 위한 설정

<div class="number-input" data-min="1" data-max="" data-step="1" data-value="1">
</div>
데이터를 1부터 시작하게 최소값 설정 최대값 무제한 1씩 증가 
data-max="10" 이면 10 이상 안올라감 = 수량제한

------------------------------------------------------------------------------------------

JS
value - step >= min // 수량을 0개로 정할 수 없게 1을 최소값으로 정해둠


perspective: 700px; //  공간감, 지름만큼 준다, 해당요소 부모요소에 준다.

transition: all 0.3s; // 해당요소에 준다.
